# Documentation Templates

## Email to Customer/Colleague (Ahmed's Style)
```
Hi [Name],

[1 sentence: what this is about]

[2-3 sentences: key deliverables/findings with numbers]

Attachments:
- [file1] — [1-line description]
- [file2] — [1-line description]

[1 sentence: next step or ask]

Thanks,
Ahmed
```

```
# [Assessment Name] — Summary

## What Was Done

## Key Findings

```

```
📊 [Project] Status — [Date]
```
